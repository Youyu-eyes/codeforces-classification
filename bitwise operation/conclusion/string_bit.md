ASCII 字符的性质：

对于大写英文字母：二进制从右往左第 6 个比特一定是 0。
对于小写英文字母：二进制从右往左第 6 个比特一定是 1。
对于任何英文字母：小写字母的二进制低 5 位，等于其大写字母的二进制低 5 位。

'A' = 65 = 0100 0001
'a' = 97 = 0110 0001

``` go
// 小写转大写：清除第 6 位
c := 'a'
upper := c &^ 32   // 'A'

// 大写转小写：设置第 6 位
c = 'A'
lower := c | 32    // 'a'

// 切换大小写：翻转第 6 位
c = 'A'
toggled := c ^ 32  // 'a'
toggled = 'a' ^ 32 // 'A'

// 字符的序号
c & 31

// 判断大小写
c >> 5 & 1
```
